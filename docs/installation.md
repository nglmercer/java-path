# Installation library

To install `java-path`

```bash
# npm install
npm install java-path

# bun install
bun install java-path

# yarn install
yarn add java-path
```

## Requirements

- **Runtime**: Node js v20.0.0 or later, Bun v1.0.0 or later
- **OS**: Windows, macOS, Linux, or Android (via Termux).
